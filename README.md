# 欢迎来到学习笔记 👋

这里按 **前端 / 后端 / 人工智能** 三大类整理学习笔记，每个分类下放具体的子单项，文章采用"核心要点 + 小结"的精简风格，方便快速复习与回顾。

## 分类导航

### 🖥️ 前端
HTML、CSS、JavaScript、Vue、React 与工程化工具等基础与进阶内容。

### ⚙️ 后端
Java、Python、MySQL、Redis、分布式系统设计等。

### 🧠 人工智能
机器学习、深度学习、大语言模型（LLM）、计算机视觉等。

## 如何运行

方式一（推荐，需 Node）：

```bash
npx docsify-cli serve .
# 访问 http://localhost:3000
```

方式二（任意带静态服务的环境）：

```bash
python3 -m http.server 3000
# 访问 http://localhost:3000
```

> 提示：首次进入会看到左侧侧边栏与顶部导航，可在 `_sidebar.md` / `_navbar.md` 中调整。
