# Redis

> 高性能内存键值数据库，常用于缓存、分布式锁、排行榜等。

## 学习要点
- 基础：5 种数据结构、过期策略、淘汰策略（LRU/LFU）。
- 持久化：RDB、AOF、混合持久化。
- 高可用：主从、哨兵、Cluster 集群。
- 进阶：Lua 脚本、分布式锁、Redis Stream、慢查询分析。
