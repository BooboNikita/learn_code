# 计算机基础

> 计算机基础相关的核心知识：网络（OSI/TCP-IP、CDN）、操作系统（程序运行、启动、进程切换、内存管理、malloc）等。

## 学习要点
- **网络模型**：OSI 七层模型、TCP/IP 四层模型、各层协议、跨层通信流程。
- **CDN**：避远就近的分布式缓存、智能 DNS 解析、多层回源、防盗链机制。
- **进程管理**：进程切换的本质（控制权转移）、并发与并行的区别、上下文切换。
- **内存管理**：虚拟内存、堆区/栈区、malloc 内存分配器（header 自描述 + 分配策略 + 拆分 + 合并）。
- **程序运行**：可执行文件格式（ELF）、加载过程、CPU 入口地址。
- **操作系统启动**：BIOS/UEFI、MBR、一阶段 + 二阶段 boot loader、PID 1 拉起。
