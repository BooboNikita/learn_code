Java、Python、MySQL、Redis、分布式系统设计等。

## 本地运行

方式一（推荐，需 Node）：

```bash
npx docsify-cli serve .
# 访问 http://localhost:3000
```

方式二（任意带静态服务的环境）：

```bash
python3 -m http.server 3000
# 访问 http://localhost:3000
```
