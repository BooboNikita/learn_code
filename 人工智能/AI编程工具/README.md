# AI 编程工具

本分类整理主流 AI 编程辅助工具的对比分析，覆盖 Agent 型编码工具、代码补全插件、IDE 扩展等方向。

## 已收录内容

- [Claude Code 与 Codex 对比分析](ClaudeCode与Codex对比.md)

## 核心关注点

1. **产品定位**：Agent 型工具 vs 代码补全插件，本地执行 vs 云端沙箱。
2. **能力边界**：上下文长度、多文件编辑、命令执行、测试运行、错误恢复。
3. **成本效率**：订阅模式、Token 消耗、API 定价。
4. **适用场景**：复杂重构、批量并行、终端自动化、前端开发等。
